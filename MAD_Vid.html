Buy & Sell Second-Hand Clothing with Style and Ease

About the App

This mobile application is designed for users to buy and sell pre-loved fashion items. Built with Flutter, it supports features such as product uploads with images, price filters, search functionality, secure login/signup, real-time sync, and seamless UI/UX tailored for a fashionable digital experience.

The video demo presents key functionalities including browsing listings, uploading clothing items, viewing seller profiles, and placing orders—all in an intuitive mobile interface.
